Memory Sculpture

Public Art Proposal

03 / 2023

《此時彼刻》

本作以三組形式分別座落於景美女中校園內戶外空間，以及景塘中央。外型取用與修澤蘭建築師所設計的校園建築造型元素——螺旋梯與弧形廊道的流動曲線，以及斜屋頂的花瓣、傘狀屋頂結構，設計出三組公共藝術裝置：《伴》、《年華》、《掛念》。三組作品皆以優雅詩意的慕白色為主要色調，兩兩一組的弧線構成對稱性，並以作品曲線建立由視覺引導的移動經驗，呼應修建築師作品的表現手法。
