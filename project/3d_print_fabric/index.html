3D列印胸花

元智大學30周年校慶 - 胸花設計與製造

Testing and researching the possible structure in 3D printing on fabric

03 / 2019

別於一般對於胸花的想像，我們設計可以讓人自行組裝的胸花，並用數位製造的方式自行生產。

本次的案件上，我們想挑戰3D列印於軟性的布料上面，選擇使用彈性布，並希望測試出一合適的平面造型，列印於彈性布上，取下後因彈性布之彈性拉力，而形成特殊的立體構型，作為我們胸花的樣態。另外，因挑戰自己大量製造的關係，流程必須非常緊湊且準確，在本次的案件上，也特別設計了快速且容易替換彈性布的新列印檯面，加裝於3D列印機原本的檯面上，並特別思考整體製造流程的規劃，使製造流程加快、更加有效利用時間。

以下幾點為本次主要的測試項目：不同彈性布材質、列印於彈性布上之列印噴頭高度、列印於彈性布上之平面造型高度、以及列印於彈性布上之平面造型設計。
