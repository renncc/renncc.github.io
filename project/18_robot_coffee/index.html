Robot Arm Coffee Show

元智大學30周年校慶 - 機器手臂咖啡秀

Robot arm motion programming design

03 / 2019

用Rhino Grasshopper為機器手臂設計一連串泡咖啡的動態，讓它在元智大學30周年校慶參訪時，可以表演機器人手沖咖啡的技能，並讓參訪的長官們可以享用咖啡。

這支KUKA機器手臂小白，於2018年加入林楚卿數位構築實驗室，讓我在2018年底至2019初擔任專案設計師的這段期間，有機會學習這有趣的新玩具，而楚卿老師也特別邀請了機器手臂大師 - 鄭基立老師來指導，而我也與實驗室的夥伴去中原建築旁聽他的機器手臂課程，了解機器手臂之基礎操作原理，而這是我第一次接觸機器手臂。

本次的動態表演，我從原本的零基礎，到可以獨自編程、操作、測試機器手臂，過程中的學習與挑戰的確不容易，但也讓我覺得相當有趣、收穫滿滿。
